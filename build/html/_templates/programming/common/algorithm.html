Data Structure And Algorithm¶

数据结构和算法是编程基础，因此虽然工作中没涉及到，我认为还是有必要复习下的。

Classic Algorithm¶

  1. 快速排序：采用分治的思想，首先寻找一个基准，然后分成两波，如此递归下去。
  2. 堆排序：采用建立堆的方法，每次建堆把堆顶元素取出来，依次来一遍即可。
  3. 归并排序：采用分治的思想，与快排思路相反，从底往上，将两个小的有序数组归并为一个大数组，之后依次递归向上。
  4. 二分查找：有序数组中进行。
  5. 线性查找（BFPRT）：将所有元素分组（每5个），查找出中位数，进行分割，与中位数进行比较，递归下去找第i大的数。
  6. 深度优先搜索（DFS）：首先访问顶点，之后访问顶点下没访问的v1点，再访问v1下没访问的v2点，之后就回溯访问其他节点。
  7. 广度优先搜索（BFS）：将根节点放入队列中，从队列中取出第一个节点访问，如不是目标，将其所有子节点放入队列中，如此继续。
  8. Dijkstra算法：最低权重（最短路径）算法，将有向权重图G分为两个集合，源点集合S/其他点集合T（包含到远点的距离）； 比较T中距离数据，将最短的临点加入到S集合中； 重新计算T集合中所有点到源点的距离，如果算上刚才的临点更短，就修改T集合中的距离值，如此继续，直到所有点都进入S集合。
  9. 动态规划算法：将原问题分解为若干子问题求解。并将计算好的子问题存储，以便快速解决重复子问题。（没用过哦）
  10. 朴素贝叶斯分类算法：在各种条件的存在不确定，仅知其概率的时候完成推理决策，是基于独立假设的。（不理解）
